# God-Script

The "holy inscriptions" found at relay sites are actually god-script — hexadecimal
code that interfaces with the [Domain](the-domain.md).

## Who can read it

| Who | Capability |
|---|---|
| [The Twelve](../04-divine-beings/the-twelve/_index.md) | Can read and **partially** write god-script |
| Minor gods | Cannot |
| [Architects](../05-empowered-humans/architects.md) | Recognize patterns without understanding the underlying language |
| Most humans | See sacred symbols, nothing more |

## Dangerous exceptions

- [Calister](../04-divine-beings/the-twelve/calister.md) (The Turning Gear) has independently figured out that god-script is a programming language and has written experimental scripts.
- Theorist Prime [Callum Vance's](../05-empowered-humans/legendary-figures/callum-vance.md) preserved notes contain passages in god-script he shouldn't have been able to write.
- The [Cabal](../09-power-and-conflict/secret-societies/the-cabal.md) has accumulated significant god-script knowledge but lacks the permissions to make it function fully.
