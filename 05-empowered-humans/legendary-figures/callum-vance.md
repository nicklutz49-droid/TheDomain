# Theorist Prime Callum Vance

Created the mathematical framework still used to describe
[Flux](../../02-the-domain/flux.md) behavior, ~1,700 years ago — every working
[Architect](../architects.md) for seventeen centuries has built on his math.

## The anomaly (restricted)

His preserved notes contain passages in
[god-script](../../02-the-domain/god-script.md) **he shouldn't have been able to
write**. Only the Twelve can partially write god-script; Architects merely recognize
patterns.
