# Drafts

Workspace for stories, outlines, and scenes set in the Domain.

Suggested conventions:
- One subdirectory per story/campaign: `drafts/<story-name>/`.
- Keep story-local inventions (new NPCs, places) in the draft until they're worth
  canonizing — then promote them into the vault via the
  [templates](../../00-meta/templates/) with `status: draft`, and record any canon
  impact in [continuity-notes](../../00-meta/continuity-notes.md).
